HTML Entities
Web sayfalarında özel karakterlerin doğru bir şekilde gösterilmesini sağlamak amacıyla kullanılan özel kodlardır. Bazı kavramları (<,>,& vb.) özel anlamlarla kullanılır, bu yüzden bu karakterlerin metin olarak görünmesini sağlamak için HTML entity'leri kullanılır.

Bir HTML entity genellikle & ile başlar ve ; ile biter.
< => < > => > ' => ' < => < & => & " => " © => Copyright sembolü

&#number; ondalık Unicode sayısı &#xnumber; genellikle başında sıfırlar bulunan onaltılık Unicode sayısı
